如何本地安装Hexo静态网站

发表于2021-01-13|更新于2021-01-13
|阅读量:

第一步[准备环境]

下载安装node.js稳定版

下载地址（https://nodejs.org/zh-cn/）
作用：创建运行环境，让JavaScript运行在服务端的开发平台
打开cmd命令行:node -v
测试是否成功

下载安装git

下载地址（https://git-scm.com/）
git是什么?开源的分布式版本控制系统，可以有效、高速地处理从很小到非常大的项目版本管理。
下载安装之后，在cmd命令行输入：git 如果输出很多关于git的参数则说明安装成功
关于git如何使用建议百度

第二步[安装Hexo]

安装的Hexo
npm install hexo-cli -g
(如果无法使用npm，说明您环境没有安装完整;或者系统无法使用npm建议使用cnpm进行安装)
cnpm安装指令
npm install -g cnpm --registry=https://registry.npm.taobao.org
初始化安装Hexo文件(输入指令后会在该终端所在位置下载文件)
hexo init "博客名"
安装依赖
npm install

第三步[测试是否安装成功]

本地运行指令
hexo s
终端显示:
INFO Start processing
INFO Hexo is running at http://localhost:4000 . Press Ctrl+C to stop.
说明Hexo已经成功运行在本地端口4000 Ctrl+C键可以推出本地运行

到此，Hexo本地安装已经成功
